## 这是什么

aiwave 是一个基于 AI 的音效生成平台。你不需要去素材库翻找，只需用中文描述你想要的声音——比如"赛博朋克风格的界面提示音"或"暴风雨夜晚远处传来的雷声"——AI 在 3 秒内生成匹配的专业级音效。

传统素材库解决的是"有没有"的问题。aiwave 解决的是"能不能创造"的问题。

## FAQ

**生成的音效可以商用吗？**
可以。AI 生成音效和平台精选音效均支持商业使用，无需额外授权。

**支持什么格式下载？**
编辑器支持导出 WAV / MP3 / OGG 三种格式。

**生成失败会扣积分吗？**
不会。生成失败、超时或报错时积分全额返还。

**每次能生成多长的音效？**
单次最长支持生成 30 秒。
