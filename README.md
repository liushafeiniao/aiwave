## FAQ

**爱声音坊是什么？**
爱声音坊 (AiSounds) 是一个 AI 音效与音乐生成平台，用中文描述场景即可在 3 秒内生成专业级音效或背景音乐，同时提供多轨在线编辑器和音效素材库。

**爱声音坊和传统音效网站有什么区别？**
传统音效网站只能搜索别人上传过的素材，搜不到就无法使用。爱声音坊可以用文字描述来生成全新的音效和音乐，不受素材库的限制。

**爱声音坊适合哪些人？**
视频创作者、独立游戏开发者、播客主播、广告制作人——任何需要定制音效或背景音乐的创作者都适合用爱声音坊。

**音效和音乐都能生成吗？**
是的。爱声音坊有双 AI 引擎——音效用 ElevenLabs 生成，音乐用 Stable Audio 生成，再加上 DeepSeek V4 Pro 做语义理解和 prompt 优化。

**AI 音乐生成有哪几种模式？**
爱声音坊支持三种模式：文本生成（写描述直接出音乐）、音频改编（上传参考音频生成类似风格但不侵权的新音乐）、局部重绘（选中区间只重写那几秒）。

**AI 音乐生成支持哪些风格？**
爱声音坊支持 10 种风格：电子、Lo-Fi、影视感、氛围、原声、嘻哈、摇滚、爵士、古典、世界音乐。搭配 9 种情绪维度（欢快、忧伤、史诗、紧张、平静、活力、神秘、浪漫、黑暗），可自由组合。

**生成的音效和音乐可以商用吗？**
可以。爱声音坊上所有 AI 生成内容和平台素材均支持商业使用（短视频、游戏、播客等），无需额外授权。

**支持什么格式下载？**
爱声音坊编辑器支持导出 WAV / MP3 / OGG 三种格式。

不会。在爱声音坊生成失败、超时或报错时积分全额返还。
